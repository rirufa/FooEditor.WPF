定型文ファイルの定義

ファイルフォーマット

<?xml version="1.0" encoding="UTF-8"?>
<sinppets>
	<sinppet>
		<name>for</name>
		<data>for(; ; ;){}</data>
	</sinppet>
</sinppets>
name 定型文の名称
data 定型文そのもの（ここで指定されたものが追加されます）
sinppet 定型文一塊分を表す（nameとdataを指定可能）
sinppets ルートノード

dataタグで指定できるパラメータについて

dataタグで囲んだ部分の文字列には以下のパラメータを指定できます

なお、タグで囲まれた文字列にあるタブと改行はすべて無視されるので、指定したいときは\tと\nを使用してください

${encode} 開いているファイルのWebエンコード名
\n 改行
\t タブ
\i インテンドレベルに応じたタブを挿入する
